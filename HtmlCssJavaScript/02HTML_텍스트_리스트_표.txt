[태그(Tag)]
<>로 표시하며 일반적으로 여는 태그(<>)와 닫는 태그(</>)를 한 쌍으로 사용

내용(content) : 여는 태그와 닫는 태그 사이에 작성

요소(Element) : 태그(tag)와 내용(content)을 합쳐서 요소라 부르지만, 보통 태그를 요소라 부름

속성(attribute)과 속성값(value) : 속성="값" 형태로 태그의 확장 기능을 해줌

이러한 요소를 이용하여 웹 문서를 작성하는 것을 Markup이라고 함


[시맨틱 태그(Semantic Tag)]
HTML의 구조를 설계하는데 있어 태그에 의미를 부여함으로써 웹사이트의 구조를 파악하기 쉽도록 도와주기 위해 만들어진 것

<header> : 큰제목이나 소개내용, 로고 등의 정보를 담는 문서의 머리말 요소

<nav> : navigation 표현을 위한 태그로 동일한 사이트 안의 문서나 다른 사이트의 문서를 연결하는 메인 메뉴 링크에 활용

<main> : 해당 페이지의 메인 콘텐츠를 나타낼 때 사용하며 반드시 한 번만 사용되어야 함. 주로 <section>, <article>, <aside> 등의 태그를 포함

<section> : 문서 내에 의미가 같은 내용들을 묶어주는 그룹 요소. 구체적인 시맨틱 태그가 없는 문서의 독립적인 영역을 나타냄

<article> : 웹 페이지 상에서 실제 내용을 담당하는 태그로 독립적으로 배포 또는 재사용이 되도록 인식하는 문서

<aside> : 간접적으로 문서와 관련된 내용을 나타내며 주로 사이드바로 사용

<footer> : 회사의 정보, 저작권 정보, 사이트 이용약관 등의 정보를 담는 문서의 꼬리말 요소


[텍스트 태그]
<hn> : 문서의 제목을 나타내는 태그. <h1>이 가장 크고 <h6>이 가장 작음. 각 웹사이트의 내용은 <h> 태그를 통해 키워드를 수집하기 때문에 <h> 태그를 이용하면 검색엔진 노출도가 높음

<p> : paragrapy. 하나의 문단을 의미하는 태그로 텍스트의 단락을 만들어줌. 단락이 끝난 이후에는 줄바꿈과 함께 약간의 줄여백이 생김

<br> : break. 줄 바꿈. 닫는 태그 없음

<blockquote> : 여백으로 인용문 표시

<strong> : bold. 웹 접근성(Web Accessibility)

<b> : bold

<em> : emphasis. italic. 웹 접근성(Web Accessibility)

<i> : italic. 기울어진 글씨로 표시. 관용구나 마음 속 생각, 기술적인 용어 등을 강조하고 싶을 때 사용

<addr> : 점선으로 밑줄을 그은 것으로 표시. 줄임말을 나타낼 때 사용. title 속성을 사용하여 전체 내용을 표시

<cite> : italic. 인용문의 출처를 나타낼 때 사용

<code> : 고정폭 글꼴로 표시. 컴퓨터 인식을 위한 소스 코드

<samll> : 부가 정보처럼 작게 표시해도 되는 텍스트

<sup> : 위 첨자를 표현하는데 사용

<sub> : 아래 첨자를 표현하는데 사용

<s> : 취소선을 그은 것으로 표시. <del>과는 다르게 삭제의 의미는 아니고 정확하지 않은 정보나 관련이 없는 텍스트를 나타낼 때 사용

<u> : underline. 밑줄을 그은 것으로 표시. <ins>와는 달리 철자 오류를 표시할 때 사용

<ins> : 밑줄을 그은 것으로 표시. 공동 작업 문서에서 삽입(insert)된 텍스트라는 의미를 부여하기 위해 사용

<del> : 취소선을 그은 것으로 표시. 공동 작업 문서에서 삭제(delete)된 텍스트라는 의미를 부여하기 위해 사용. 

<hr> : horizontal rule. 가로줄. 닫는 태그 없음

<div> : 블록 요소와 인라인 요소를 그룹으로 묶을 때 사용

<span> : 줄바꿈 없이 영역을 묶기 위해서 사용. 인라인 요소를 그룹으로 묶을 때 사용. 블록 요소는 적용되지 않음

<address> : 실제 주소뿐만 아니라 문서의 연락처 정보를 의미

<q> : 간단한 문장을 인용한 것을 나타낼 때 사용. <blockquote> 태그와 달리 줄바꿈이 되지 않고 인용 내용에 큰따옴표를 붙여서 표시

<mark> : 형광펜을 칠한 것처럼 표시. 맥락상 관련성이 있는 곳을 하이라이트 할 때 사용

<kbd> : 고정폭 글꼴로 표시. 키보드 입력, 음성 입력 등 사용자의 입력을 나타낼 때 사용

<samp> : 고정폭 글꼴로 표시. 컴퓨터 프로그램 출력의 예시 등을 나타낼 때 사용

&nbsp; : non breaking space. 문자 사이에 공백을 넣을 때 사용. 태그 없음


[목록 만들기 태그]
<ol> : ordered list. 순서가 있음
<ol type="1"> 숫자(디폴트)
<ol type="a"> 영문 소문자
<ol type="A"> 영문 대문자
<ol type="i"> 로마자 숫자 소문자
<ol type="I"> 로마자 숫자 대문자
<ol start="n"> n번째 번호부터 시작

<ol>
	<li></li>
	...
	<li></li>
</ol>

<ul> : unordered list. 순서가 없음
<ul type="disc"> : 검은색 원(디폴트)
<ul type="circle"> : 빈 원
<ul type="square"> : 사각형

<ul>
	<li></li>
	...
	<li></li>
</ul>


<li> : 순서가 있는 목록이나 순서가 없는 목록의 각 항목을 나타냄

<dl> : definition list. 설명 목록의 시작과 끝을 나타냄
<dt> : definition title. 설명 목록의 이름(제목)을 나타냄
<dd> : definition data. 설명 목록의 값(설명)을 나타냄

<dl>
	<dt></dt>
	<dd></dd>
<dl>


[표 만들기 태그]
table : 표
row 행
column 열
cell 셀

<table> : 표 전체
<caption> : 표 제목
<tr> : 행
<td> : 셀
<th> : 제목 셀
<thead> : 표 구조에서 제목 부분
<tbody> : 표 구조에서 본문 부분
<tfoot> : 표 구조에서 요약이나 정리 부분
<col> : 표에서 열의 스타일을 지정
<colgroup> : 표에서 여러 열을 함께 묶어서 스타일을 지정
 
<table>
	<caption></caption>
	<tr>
		<td></td>
		...
		<td></td>
	</tr>
	...
	<tr>
		<td></td>
		...
		<td></td>
	</tr>
</table>
