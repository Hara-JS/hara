[폼(form)]
html에서 사용자가 입력할 수 있는 양식을 제공하는 태그. form 태그 내에 input 태그들을 통해 사용자가 입력한 정보를 서버로 넘기면서 요청하는 역할


[폼에서 사용하는 태그]
<form> : 폼을 만드는 태그
	method : get, post 방식으로 전송 방법을 지정
	get : 서버에 사용자가 입력한 값들을 URL에 노출시켜서 전달하는 방식
	post : 입력 내용의 길이에 제한을 받지 않고, 서버에 사용자가 입력한 값들을 URL에 노출시키지 않고 전달하는 방식. 서버가 존재해야 함
	name : 폼의 이름. 한 문서 안에 여러 개의 폼 태그가 있을 경우 폼들을 구분하기 위해 사용. 나중에 서버로 값을 전달할 때 name 속성과 텍스트 필드에 작성된 값이 세트로 넘어감
	action : 폼 내에 입력된 값들을 전달할 서버 상의 프로그램을 지정
	autocomplete : 자동 완성 기능. 이전 입력 내용을 출력하는 기능. 각 태그별로 지정 가능 (default : on)
	target : 스크립트 파일을 현재 창이 아닌 다른 위치에 열도록 지정

<label> : 폼 요소에 레이블을 붙이는 태그

<fieldest> :  폼 요소를 그룹으로 묶는 태그

<legend> : <fieldset> 태그로 묶은 그룹에 제목을 붙이는 태그

<input> : 사용자가 내용을 입력하는 필드를 삽입할 때나 버튼을 삽입할 때 사용

<select> : 드롭다운 목록을 삽입
	size : 화면에 표시될 드롭다운 메뉴 개수를 지정
	multiple : 다수 선택 가능

<option> : 드롭다운 목록의 옵션을 삽입
	label : 표시될 옵션의 제목
	value : 서버로 넘겨질 값
	selected : 옵션이 선택 되었음을 표시

<textarea> : 여러 줄의 텍스트를 입력할 수 있는 텍스트 영역을 삽입
	cols : 가로 너비
	rows : 세로 너비 (줄 단위)

<datalist> : 데이터 목록을 삽입

<optgroup> : <option>을 그룹화하기 위해 사용
	label : 그룹 제목을 붙임


[<input> 태그의 type]
text : 한 줄짜리 텍스트를 입력할 수 있는 텍스트 박스를 삽입

password : 비밀번호를 입력할 수 있는 텍스트 박스를 삽입. 사용자의 입력 값을 확인할 수 없음

search : 검색 박스를 삽입. 텍스트 입력시 지우기 버튼 표시

url : URL 주소를 입력할 수 있는 필드를 삽입

email : 이메일 주소를 입력할 수 있는 필드를 삽입. 입력한 내용이 메일 주소 형식에 맞는지 자동으로 체크

tel : 전화번호를 입력할 수 있는 필드를 삽입

checkbox : 여러 항목 중 2개 이상 선택이 가능한 경우 사용. 태그 여러 개를 선언하여 하나의 용도로 사용할 경우 name 속성의 값이 같아야 함

radio : 여러 항목 중 하나만 선택해야 할 경우 사용. 태그 여러 개를 선언하여 하나의 용도로 사용할 경우 name 속성의 값이 같아야 함

number : 입력창에 숫자를 입력하는 것이 기본이지만 브라우저에 따라서 스핀 박스가 표시되기도 함 (스핀 박스 : 입력창 오른쪽에 위아래 화살표 버튼)

range : 숫자를 조절할 수 있는 슬라이드 바를 삽입

date | month | week | time | datetime | datetime_local : 웹 문서나 애플리케이션에 달력을 포함하고 싶을 때 사용

submit : 사용자가 폼에 입력한 정보를 서버로 전송하는 버튼을 삽입

reset : 사용자가 폼에 입력한 모든 정보를 재설정해 모든 내용을 리셋하는 버튼을 삽입

image : submit 버튼 대신 사용할 이미지를 삽입

button : 일반 버튼을 삽입

file : 파일을 첨부할 수 있는 버튼을 삽입

hidden : 사용자에게는 보이지 않지만 서버로 넘겨주는 값이 있는 필드를 생성


[<input> 태그의 속성]
autofocus : 웹 문서를 로딩하자마자 해당 텍스트 필드에 마우스 커서가 표시

placeholder : 텍스트 필드에 힌트 내용을 표시하고 있다가 그 필드를 클릭하면 사라지는 내용

readonly : 입력 필드를 읽기 전용으로 지정

required : 필수 입력 필드를 지정. 폼이 전송될 때 필수 입력 필드에 필요한 내용이 모두 채워졌는지 체크

max : 필드에 입력할 수 있는 최댓값

min : 필드에 입력할 수 있는 최솟값

step : 숫자 입력 필드에서 증감할 숫자 간격. 디폴트는 1이며 생략 가능

maxlength : 텍스트 필드에 입력할 수 있는 최대 길이를 지정

minlength : 텍스트 필드에 입력할 수 있는 최소 길이를 지정

size : 텍스트 필드에서 화면에 표시할 크기를 지정

list : 연결할 데이터 목록을 지정
